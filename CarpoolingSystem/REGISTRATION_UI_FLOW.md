## Key Design Principles

1. **Immediate Feedback**: Real-time validation as user types
2. **Clear Errors**: Inline errors next to fields + summary box
3. **Progressive Disclosure**: Show only relevant fields for selected role
4. **Visual Hierarchy**: Clear sections with dividers and headers
5. **Accessibility**: Standard UIKit text fields, clear labels
6. **Error Prevention**: Input filtering (digits-only phone), date ranges
7. **User Guidance**: Placeholder text, hints, example formats
8. **Consistent UX**: Same validation pattern for all fields
